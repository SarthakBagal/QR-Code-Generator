# QR-Code-Generator
Generating QR code for any text

A simple and responsive QR Code Generator built using HTML, CSS, and JavaScript.
This project allows users to generate QR codes instantly by entering any text or URL, using a free public QR code API.

**QR codes are generated using the following API:**
https://api.qrserver.com/v1/create-qr-code/

**How It Works:**
1. User enters a text or URL in the input field
2. Clicks on Generate QR
3. JavaScript sends the input data to the QRServer API
4. API returns a QR image
5. QR code is displayed instantly on the page

**Learning Outcome**
  1. Working with external APIs in JavaScript
  2. DOM manipulation
  3. Event handling
  4. Dynamic image generation
